About

I build trading systems and digital platforms. My clients are founders, traders, and companies who need production-ready software — not proposals. I write the code. I deploy it. I maintain it.

Latest projects: Trabro, a high-frequency market maker framework for crypto DEXs. Event-driven Rust with ZeroMQ, QuestDB, and multi-exchange connectors. And Inkt, a book library platform with audio-text sync and highlighting. Rust/Axum backend, SvelteKit frontend. Designed, built, and deployed from scratch.

Before that: AI models for a hedge fund startup. Open-source contributions to the biggest crypto software project. I've been doing this since 2018.

I work with people who value execution over process. If you need sophisticated technical infrastructure built and shipped, let's talk.
